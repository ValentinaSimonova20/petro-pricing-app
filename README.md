# Сервис расчёта цен (СИБУР × Центральный университет)

Веб-сервис для автоматического расчёта договорных цен по строкам прогноза продаж. Система подбирает формулу из каталога договоров, подставляет котировки, курсы валют и константы, считает цену и показывает расшифровку — без ручной сводки Excel и учётных выгрузок.

Проект выполняется в рамках буткемпа СИБУР × Центральный университет. Цель на первом этапе — рабочий MVP на тестовом наборе (~500 строк) с той же логикой, что у промышленного контура (~12 000 строк в месяц).

## Зачем нужен сервис

Сейчас аналитики каждый месяц вручную обрабатывают прогноз «клиент + продукт + период + объём»: ищут формулу среди тысяч записей, подставляют котировки и курсы, сводят результат из разных файлов. На реальном объёме это занимает порядка **200 часов в месяц**, легко выбрать не ту формулу или дату котировки и сложно объяснить, из чего сложилась цена.

Сервис автоматизирует цепочку: **загрузка данных → подбор формулы → подстановка значений → расчёт → расшифровка → ручная правка → выгрузка Excel**.

## Ключевой функционал

Планируется реализовать:

1. **Загрузка прогноза продаж** — таблица строк с клиентом, продуктом, периодом, объёмом и атрибутами сделки.
2. **Загрузка справочников** — каталог формул, компоненты формул, группы материалов, котировки, маппинг котировок, курсы валют, типы термов.
3. **Подбор формулы** к каждой строке прогноза по правилам области применения: клиент, группа материалов M, период действия, тип сделки; при нескольких подходящих вариантах — показ всех и выбор по умолчанию (самая поздняя дата создания).
4. **Подстановка значений компонентов** — котировки, курсы, константы/надбавки на нужную дату (приоритет типов публикации: Факт → ОФ → ППР).
5. **Расчёт итоговой цены** по текстовому выражению формулы на стороне сервера через API.
6. **Расшифровка расчёта** — какая формула применена, какие числа подставлены и из какого источника.
7. **Ручные правки** — ввод своей цены и выбор другой формулы, если вариантов несколько.
8. **KPI качества** — доля строк с найденной формулой и посчитанной ценой; доля совпадений с эталоном (допуск ±3%).
9. **Выгрузка результата в Excel** для передачи дальше по процессу.
10. **Совместная работа** нескольких пользователей над частями данных с общим сводом результатов.

## Целевая аудитория и сценарий использования

**Кто пользуется:** аналитики ценообразования / продаж, которые ежемесячно готовят расчёт цен по прогнозу спроса.

**Типовой сценарий:**

1. Открыть сервис в браузере, загрузить исходные таблицы (или выбрать уже загруженный набор).
2. Указать параметры расчёта (период, при необходимости — фильтр по продуктам / группам материалов).
3. Запустить расчёт — система подбирает формулу и считает цену по каждой строке.
4. Просмотреть таблицу результатов и сводные KPI.
5. Открыть расшифровку по строке: формула, компоненты, источники значений, признак ручной правки.
6. При необходимости исправить цену или выбрать другую формулу.
7. Скачать итоговый Excel.

На защите MVP эксперт проходит тот же сценарий самостоятельно: параметры → расчёт → расшифровки → ручные правки → выгрузка.
